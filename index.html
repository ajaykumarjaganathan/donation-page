The Project

What I'm Building 🤖

I'm Ajaykumar Jaganathan — a solo developer building a full-stack attendance system powered by a real autonomous robot that physically moves through classrooms and industrial sites.

The robot autonomously patrols spaces, uses sensors to detect and identify students and workers, and syncs all data to a cloud dashboard in real time — no human intervention needed.

Right now, I'm building both the robot hardware AND the full-stack software solo, with zero budget. Every rupee goes into robot parts, sensors, hosting, and making this vision real.
